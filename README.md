# Pact Night Sight Illumination
This mod allows the player to toggle reticle illumination in the TPN-1 series of night sights, just like the day sights, using the [ I ] key by default. This will apply to the T-54/55, T-62, T-64 and T-72 variants. 

**Installation**  
Melonloader is required, simply place the DLL file into the /Mods/ folder.
